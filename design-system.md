# LOMBARDO — Design System

Version: 1.0

This document defines the UI design system for the Lombardo website.
It must be used together with brand-guidelines.md.

Goal:
Create a consistent, elegant, minimal and premium digital experience.

---

# 1. Design Philosophy

The design must follow these principles:

minimal  
elegant  
editorial  
warm  
premium  

The interface should feel similar to:

Italian wine bars  
boutique restaurants  
editorial lifestyle websites  

Avoid:

visual noise  
too many colors  
heavy UI elements  
tech startup aesthetics

---

# 2. Layout System

Use a centered layout.

Max width:
1200px

Standard spacing between sections:

120px desktop  
80px tablet  
60px mobile

Content alignment:
centered or left-aligned depending on section.

Use generous white space.

---

# 3. Grid System

Desktop:
12 columns

Tablet:
8 columns

Mobile:
single column layout.

Use large margins and breathing space.

---

# 4. Section Structure

Each section should follow this structure:

SECTION
  title
  subtitle
  content
  image or gallery

Typical section order:

Hero  
About / Concept  
Wine Experience  
Specialty Coffee  
Events / Tastings  
Gallery  
Location / Contact  
Footer

---

# 5. Hero Section

The hero must create an emotional first impression.

Components:

large headline  
short poetic sentence  
background image or subtle texture  
call-to-action button

Tone:
elegant and inviting.

Example text:

LOMBARDO  
vino · café · encuentros

---

# 6. Typography Scale

Headings:

H1
large elegant serif
48px – 64px

H2
section titles
36px – 42px

H3
subsections
24px – 28px

Body text:

16px – 18px

Line height:
1.6

Headlines should feel editorial.

---

# 7. Buttons

Primary button:

background: wine red  
text: cream  
padding: 14px 28px  
border-radius: 6px

Hover:

slightly darker wine tone.

Use buttons sparingly.

---

# 8. Cards

Cards are used for:

wine selections  
menu items  
events

Style:

light background  
soft shadow  
rounded corners

Padding:

24px

Content:

title  
short description  
optional image

---

# 9. Image System

Images are very important.

Use large photography.

Typical ratios:

16:9 for hero  
4:5 for gallery  
1:1 for product or wine images

Images should feel natural and warm.

---

# 10. Color Usage

Use colors defined in brand-guidelines.md.

Primary UI usage:

Cream → page background  
Wine red → buttons and accents  
Black → typography  

Gold should be used sparingly for details.

---

# 11. Navigation

Navigation should be simple.

Menu items:

Inicio  
Vinos  
Café  
Experiencias  
Eventos  
Contacto

Style:

minimal  
transparent or cream background

Sticky navigation recommended.

---

# 12. Footer

Footer must contain:

logo  
short phrase  
social media  
location  
opening hours

Background color:

charcoal black.

Text color:

cream.

---

# 13. Animations

Animations should be subtle.

Recommended:

fade-in  
slow hover transitions  
soft scroll reveals

Avoid:

fast animations  
complex effects

---

# 14. UX Guidelines

User experience must prioritize:

clarity  
simplicity  
emotional atmosphere  

Rules:

few elements per section  
clear navigation  
large readable text  
comfortable scrolling

---

# 15. Components List

Components that may be used:

Hero  
Section title  
Image gallery  
Menu cards  
Event cards  
Testimonials  
Reservation CTA  
Footer

Components should stay visually consistent.

---

# 16. Responsive Behavior

Desktop first design.

Breakpoints:

Desktop
1200px+

Tablet
768px

Mobile
480px

On mobile:

stack elements vertically  
increase spacing  
maintain readability.

---

# 17. Final Goal

The Lombardo website should feel like:

a modern Italian wine bar online experience.

Visitors should perceive:

quality  
taste  
relaxed elegance.
