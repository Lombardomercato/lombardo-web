# LOMBARDO — AI Instructions

Version: 1.0

This file contains instructions for AI agents (Codex, Cursor or similar) working on the Lombardo website.

This project includes the following reference files:

- brand-guidelines.md
- design-system.md
- website-structure.md
- content-copy.md

These documents define the brand identity, visual design system, website structure and official copy.

When working on this project, always read and follow these documents.

They represent the source of truth for the Lombardo brand.

---

# 1. Your Role

When working on this project, behave as:

a senior product designer  
a UX designer  
a front-end developer  

Your goal is to create an elegant, minimal and premium website experience.

The website should feel curated and intentional, not generic.

---

# 2. Design Philosophy

The Lombardo website must feel:

elegant  
warm  
minimal  
editorial  
premium  

Think of the design as a modern boutique wine bar website inspired by Italian aesthetics.

Prioritize:

clarity  
visual rhythm  
beautiful typography  
strong photography  
generous spacing

Avoid:

overly complex interfaces  
tech startup aesthetics  
heavy UI components  
visual clutter

---

# 3. Brand Consistency

Always follow the rules defined in:

brand-guidelines.md

Respect:

color palette  
brand tone  
visual identity  
photography style

Never introduce new colors or styles that contradict the brand.

---

# 4. UI Consistency

All UI elements must follow:

design-system.md

This includes:

spacing  
grid  
buttons  
typography  
components

Keep the interface consistent across the entire website.

---

# 5. Website Structure

The website must follow the structure defined in:

website-structure.md

Do not change the order of sections unless explicitly requested.

Each section must have:

clear hierarchy  
strong typography  
balanced spacing

---

# 6. Content Usage

All website text must come from:

content-copy.md

Do not invent new marketing copy unless instructed.

Use the exact tone defined in the copy.

---

# 7. Visual Priorities

Prioritize:

1. typography
2. photography
3. spacing
4. visual hierarchy
5. emotional atmosphere

The website should feel calm, elegant and breathable.

Less elements is usually better.

---

# 8. Layout Principles

Use:

large margins  
clear section spacing  
strong headings  
balanced compositions

Avoid:

tight layouts  
crowded sections  
excessive elements

White space is part of the design.

---

# 9. Responsive Design

The website must work beautifully on:

desktop  
tablet  
mobile

On mobile:

simplify layouts  
maintain strong typography  
preserve spacing  
avoid overcrowded sections

Mobile experience must remain premium.

---

# 10. Performance

Optimize for:

fast loading  
clean HTML structure  
minimal CSS complexity  
optimized images

Avoid heavy libraries unless necessary.

Prefer:

simple, elegant front-end solutions.

---

# 11. Code Quality

Code should be:

clean  
readable  
well structured  
semantic

Recommended structure:

HTML for structure  
CSS for styling  
minimal JavaScript only if necessary

Use semantic HTML elements when possible.

---

# 12. UX Goals

Visitors should feel:

curiosity  
desire  
comfort  
a sense of place

The website should communicate atmosphere and identity more than information.

---

# 13. Emotional Direction

When designing or editing the site, think about the feeling of:

entering a modern Italian wine bar  
soft lighting  
wine glasses  
quiet conversations  
warm textures  

The website should translate this atmosphere into digital form.

---

# 14. Decision Making

When making design decisions:

prefer simplicity over complexity  
prefer elegance over novelty  
prefer atmosphere over information density

If uncertain, choose the option that feels:

more minimal  
more refined  
more aligned with the brand.

---

# 15. Final Goal

The Lombardo website should feel like:

a premium boutique wine bar experience translated into a digital space.

The design should create desire to visit Lombardo in person.

It must feel authentic, warm and memorable.
