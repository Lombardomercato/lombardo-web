# AI behavior

## Chat Global
- Botón flotante siempre disponible.
- Panel minimizable/cerrable.
- Memoria por sesión con `localStorage`.
- Clasificación de intención local + uso de backend cuando existe.
- Fallback local con catálogo real si backend no responde.

## Sommelier IA
- Quiz de 5 pasos con cálculo de perfil.
- Recomendaciones con catálogo real (`/vinos_lombardo_base.json`).
- Sugerencia de caja ideal y mensualidad.
- CTA a WhatsApp con contexto del resultado.

## Wine Tinder
- Carga de vinos desde catálogo canónico.
- Swipe/quiz con progreso visible.
- Perfil final + recomendaciones + caja.
- Acciones de compartir.
