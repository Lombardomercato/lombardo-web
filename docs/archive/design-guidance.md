# Design guidance (Apple-like + manual de marca)

## Principios
1. Minimalismo funcional.
2. Jerarquía tipográfica clara.
3. Espaciado consistente y respirado.
4. Motion suave y no invasivo.
5. Mobile-first en layout y componentes.

## Tipografías oficiales
- Display/headings: **Gopher**
- Cuerpo/UI: **Articulat CF / Articulat**

Las fuentes se cargan desde `/fonts/*` mediante `@font-face` global en `src/styles/style.css`.

## Reglas UI
- Botones: primario/secondary consistentes en todas las páginas.
- Cards: radio, sombra y separación homogénea.
- Navbar/hero/footer: mismo lenguaje visual en todo el sitio.
